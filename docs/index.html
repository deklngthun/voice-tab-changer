Setup Guide

Up and running in 2 minutes.

Step-by-step for both platforms.

  1. Open Terminal and paste one command

    Press Cmd + Space, type Terminal, hit Enter. Then paste:

    curl -fsSL https://deklngthun.github.io/voice-tab-changer/install.sh | bash
    💡 This downloads, installs, and removes the security warning automatically. No "malware" pop-up.
  2. Grant Microphone permission

    macOS will show a dialog: "VoiceTabChanger wants to access the microphone". Click Allow.

  3. Grant Accessibility permission

    Go to System Settings → Privacy & Security → Accessibility. Find VoiceTabChanger and toggle it on. This lets the app switch windows on your behalf.

  4. You're done! 🎉

    A V icon appears in your menu bar. The app is listening. Say any app name — "Chrome", "Spotify", "Finder" — and it will switch or launch it instantly.

  1. Download & unzip

    Click Download for Windows above. Right-click the downloaded .zip → Extract All.

  2. Run the app

    Double-click VoiceTabChanger.exe.

  3. Bypass SmartScreen

    Windows may show a blue "Windows protected your PC" dialog. Click More info → Run anyway.

    💡 Why? Microsoft requires an expensive code-signing certificate for this warning to disappear. The app is safe.
  4. Allow Microphone access

    Windows will ask for microphone permission. Click Yes / Allow.

  5. You're done! 🎉

    A V icon appears in your system tray (bottom-right). Hold Ctrl + Shift + Space, say an app name, and release. The window switches instantly.
